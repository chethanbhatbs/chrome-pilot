# Screenshots

Drop PNG/GIF files here. The main README expects these filenames:

- `hero.gif` — 30-second demo loop (open sidebar → Cmd+K search → focus mode). Keep under 5 MB.
- `sidebar.png` — main sidebar with tab tree
- `command-palette.png` — Cmd+K palette open with results
- `focus-mode.png` — focus session active with blocked-action overlay
- `workspaces.png` — workspaces panel with custom icons/colors

Recommended size: 1280×800 (or 2x for retina). PNG-8 or optimized PNG-24.
